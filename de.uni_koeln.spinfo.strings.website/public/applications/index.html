There have been three experimental applications of suffix trees for natural language within coursework for higher-level courses conducted at the chair for Linguistic Information Processing of the Department of Linguistics, University of Cologne, Germany. These include:

  • Morphological analysis of spanish texts (paper in german, code included in latest release and via Subversion in project "Morphologische Analyse")
  • Approximate matching and general applicability of suffix trees in NLP (paper in german, code included in latest release and via Subversion in project "de.uni_koeln.spinfo.strings.core"). Included are implementations of the algorithms for matching with k-mismatch and wildcards as described in Gusfield 1997:196f. as well as experimental implementations of constant-time LCA-retrieval (based on implementations by A.G. McDowell)
  • Text classification with paradigms created using suffix trees (in german, the download includes Java source code, an executable Jar and required files; it will execute a test run and write results to a file. The code can also be accessed via Subversion, in project "de.uni_koeln.spinfo.is.document_tagger")

We are working on an integration of suffix trees for natural language processing into Tesla, the Text Engineering Software Laboratory, which is currently under development in our department. Our goal is making suffix trees available as a component for building and experimenting with NLP systems.
